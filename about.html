Welcome to my portfolio site!

My name is Dan Ewing.
I'm an inspiring level designer and artist. I've been working with games and game engines since 2017.
For most of my time, and currently all I have to show for my portfolio so far, I've been using the Source Engine.
To be more specific, I use the level editor of Valve's first-person-shooter; Counter Strike: Global Offensive.
Source is a 3D game engine developed by Valve Software and used to make games for the Half-Life, Team Fortress, Counter Strike, and other series.
